Запуск игры устный счет
1. В командной строке зайти в директорию проекта GameVerbalCount
2. Запустить jar файл командой java -jar target/GameVerbalCount-1.0-SNAPSHOT.jar
3. В адресной строке браузера набрать http://localhost:8080/
4. Можно играть без регистрации или зарегистрираваться, тогда можно будет сравнить свои результаты
с другими игроками
